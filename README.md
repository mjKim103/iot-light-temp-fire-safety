# iot-light-temp-fire-safety
IoT 기반 조도/온도 스마트 방재 시스템
📌 1. 프로젝트 개요

본 프로젝트는 Arduino 기반 환경 센서를 활용하여 화재 발생 시 감지되는 온도 변화 및 조도 변화를 실시간으로 분석하고 위험 상황일 경우 즉각적으로 경보 음을 출력하는 IoT 방재 시스템이다.

화재는 일반적으로 열 상승과 빛 변화(또는 빛 차단)를 동반하므로, 본 프로젝트에서는 연기 센서가 없는 상황에서도 조도 + 온도 조합을 통한 화재 위험 판단이 가능하도록 설계하였다.

🎯 2. 프로젝트 목표

온도 및 조도 센서를 사용한 화재 발생 징후 감지

위험 상태 시 피에조 부저를 통한 경보 기능

실시간 모니터링 기능 구현

저가형 부품을 활용한 간단한 IoT 방재 기술 실험

🔍 3. 개발 목적 및 필요성

화재는 초기 대응 여부에 따라 피해 정도가 크게 달라진다. 본 시스템은 전문 방재 센서가 없는 환경에서도 일상적인 센서만으로 위험 감지 실험이 가능하다는 점에 의의가 있다.

또한 Arduino 기반 IoT 시스템 설계 과정을 통해 센서 활용, 내장형 시스템, 경보, 데이터 판별 로직을 종합적으로 학습하고, 안전기술에 대한 IoT 적용 가능성을 탐색하였다.

⚙ 5. 시스템 동작 방식
🔥 위험 판단 조건
IF 온도 > 임계값 AND 조도 변화 > 기준값
→ 화재 의심
→ 부저 ON

🔆 조도 변화가 의미하는 것

주변이 갑자기 매우 밝아짐 = 불꽃 발생

주변이 갑자기 어두워짐 = 연기 가림 효과
